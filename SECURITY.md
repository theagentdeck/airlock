# Security Policy

**Supported Versions**

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

**Reporting a Vulnerability**

If you discover a security vulnerability in AirLock, please report it responsibly.

Email: **security@theagentdeck.ai**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We aim to acknowledge within 48 hours and resolve critical issues within 7 days. We follow a coordinated disclosure process.

**Scope**

AirLock is a security tool — the scanner and protocol. Do not use this repository to report security issues in third-party sites or services that integrate with AirLock. Report those directly to the affected parties.
